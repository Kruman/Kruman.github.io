Aplicación Móvil Para La Gestión De Los Rescatistas De Mascotas

Este proyecto se enfocó en la creación de una aplicación móvil para la Fundación Albornoz Jiménez, con el propósito de simplificar la gestión eficiente de los rescatistas y dar seguimiento a los animales en situación de vulnerabilidad. Tecnologías Utilizadas

  • Lenguaje de Programación: Kotlin
  • IDE: Android Studio
  • Metodología: Scrum
  • Funcionalidades: Generación de PDF, gestión de información de rescate, etc.

Desarrollo

Se implementó la metodología Scrum para el desarrollo de la aplicación, utilizando Kotlin y Android Studio. A lo largo de un proceso estructurado de diseño y desarrollo, se logró crear una aplicación que aborda de manera efectiva los problemas identificados en la gestión de los rescatistas y la recopilación de información de rescates.

Funciones Clave

  • Captura y administración de información de rescate
  • Generación de documentos PDF para el seguimiento y análisis de casos
  • Mejora de la eficiencia y la coordinación en la fundación

PC Era - Tienda de Componentes de Computadora y Oficina

El proyecto "PC Era" consistió en el desarrollo de una página web y una aplicación móvil para una tienda especializada en la venta de componentes de computadora y artículos de oficina. El objetivo principal fue crear una plataforma intuitiva y fácil de usar para facilitar las compras de los usuarios.

Tecnologías Utilizadas

  • Lenguaje de Programación: JavaScript (para la web) y Kotlin (para la aplicación móvil)
  • Frameworks y Herramientas: HTML, CSS, Android Studio
  • Metodología: Scrum

Desarrollo

El proyecto se llevó a cabo utilizando la metodología Scrum para asegurar un desarrollo ágil y colaborativo. La página web fue desarrollada con HTML, CSS y JavaScript, mientras que la aplicación móvil fue creada con Kotlin en Android Studio. Se diseñó una interfaz de usuario intuitiva para ambas plataformas, permitiendo a los usuarios navegar y realizar compras de manera eficiente.

Funciones Clave

  • Registro y gestión de productos, precios y cantidades
  • Carrito de compras y procesamiento de pedidos
  • Generación de PDF de las ganancias y reportes de ventas
  • Interfaz de usuario amigable y fácil de usar
  • Sistema de búsqueda y filtrado de productos

Sistema de Ventas con Java y MySQL

El proyecto consistió en el desarrollo de un sistema de ventas utilizando Java y MySQL. Este sistema incluye un formulario de inicio de sesión, la capacidad de registrar nuevas ventas, actualizar información de clientes, proveedores y productos, y generar reportes en formatos Excel y PDF. El objetivo principal fue crear una solución integral para gestionar ventas y relaciones con clientes y proveedores.

Tecnologías Utilizadas

  • Lenguaje de Programación: Java
  • Base de Datos: MySQL
  • Bibliotecas y Frameworks: JDBC, Apache POI (para Excel), iText (para PDF)
  • Entorno de Desarrollo: Eclipse o IntelliJ IDEA

Desarrollo

El desarrollo del sistema de ventas se llevó a cabo utilizando Java para la lógica de la aplicación y MySQL para la base de datos. Se implementaron funcionalidades clave como formularios de inicio de sesión, registro de nuevas ventas, y gestión de datos de clientes, proveedores y productos. Además, se integraron bibliotecas para la generación de reportes en Excel y PDF.

Funciones Clave

  • Inicio de Sesión: Formulario seguro para el inicio de sesión de usuarios.
  • Nueva Venta: Registro de nuevas ventas con detalles del producto, cantidad y precios.
  • Gestión de Clientes: Actualización y mantenimiento de información de clientes.
  • Gestión de Proveedores: Actualización y mantenimiento de información de proveedores.
  • Gestión de Productos: Actualización y mantenimiento de inventario de productos.
  • Generación de Reportes: Exportación de datos de ventas a Excel y generación de facturas en formato PDF, incluyendo información de la empresa y del cliente.
